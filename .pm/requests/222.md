# Issue #222: Feature: Capacitor Mobile App (iOS & Android) — Single Codebase

**GitHub URL:** https://github.com/grootdev-ai/groot-finance/issues/222
**Filed:** 2026-02-21
**Label:** pm:feature-request
**Status:** Open

## Summary

Wrap the existing Next.js web app in Capacitor to ship native iOS and Android apps from the same codebase. Minimal-effort path to app store distribution with 100% frontend code reuse.

## Key Phases

1. **Core Setup** — Install Capacitor, init project, add iOS/Android platforms
2. **Native Plugins** — Camera, push notifications, deep linking, status bar
3. **App Store Prep** — Icons, signing, store listings, beta testing
4. **Polish & Release** — Performance testing, offline validation, submission

## Trade-off Decision

Capacitor chosen over React Native (too much UI rewrite), Flutter (separate codebase), and PWA-only (no app store, limited native APIs).
